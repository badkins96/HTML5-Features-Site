The W3C is now leading the charge for many web technologies:

Mission Statements & Goals

WebRTC

Provides browsers Real-Time Communications via simple APIs

Web Assembly

Binary instruction format for a stack-based virtual machine, allowing deployment on the web for client and server applications by being a portable target for compilation of high-level languages

WebPerf

Provide methods to observe and improve aspects of application performance of user agent features and APIs

Web Payments

Make payments easier and more secure on the Web

WebVR

Make it easier for everyone to get into VR experiences, no matter what device you have

WebAuthn

Standardize an interface for authenticating users to web-based applications and services

Service Worker

Enable applications to take advantage of persistent background processing, including hooks to enable bootstrapping of web applications while offline

MSE

Allow JavaScript to generate media streams for playback
